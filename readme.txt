Ce template vous donne une base pour élaborer votre rapport de PFE. Il n'est pas parfait, vous pouvez l'adapter et l'améliorer. N'hésitez pas à me faire un retour sur les bugs et les améliorations apportées:yannis.leguennec@phelma.grenoble-inp.fr.

Attention, c'est le fichier "Rapport_PFE.tex" (i.e. le main) qui doit être compilé. La compilation des autres fichiers produira une erreur. 

Le fichier "latexmkrc" est indispensable pour générer correctement le glossaire (c'est une erreur connue de Overleaf qui nécessite ce correctif)

Le fichier "Settings.tex" définit tous les packages nécessaires pour le rapport, la mise en forme des pages...

Le fichier "glossary.tex" vous permet de définir les acronymes, la table des acronymes se mettant automatiquement à jour.

Le fichier "biblio.bib" contient les références biliographiques à mettre à jour et à introduire dans le texte.

Le fichier "gantt.text" contient un diagramme de Gantt que vous pouvez adapter à votre projet.






N'oubliez pas qu'il y a également un rapport de compétences à fournir, un autre template est disponible pour celui-ci.
